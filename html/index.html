  • Autocomplete search

    Search by common name, formal name, alias, or 2-letter code.

  • Locale suggestion

    Uses browser locale to surface a likely country without forcing a default.

  • Full dataset

    Ships the complete country and region list instead of a short sample set.

  • Portable structure

    HTML, CSS, JavaScript, and data are separated for easier reuse.

Files

  • index.html contains only the demo markup.
  • styles.css contains presentation and modal layout styling.
  • app.js loads country data and wires the picker behavior.
  • countries.json contains the full country and region records used by the demo.
