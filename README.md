# Better Country Select

This is demo code for a modal country select which addresses many shortcomings of common country select inputs.

GitHub:

https://github.com/talysto/better-country-select

Blog post:

https://talysto.com/blog/building-better-country-selects/

View the demo online:

https://talysto.com/blog/building-better-country-selects/

## Get the code

Clone the repository:

```bash
git clone https://github.com/talysto/better-country-select.git
```

Serve the `html` folder locally with a single command from the `html` directory:

```bash
python -m http.server 8000
```

Start with:

- `better-country-select/html/index.html`
- `better-country-select/html/styles.css`
- `better-country-select/html/app.js`
- `better-country-select/html/countries.json`
